# Zenith OS: Kernel Resource & Process Monitor
**Developer:** [Your Name]
**Course:** Operating Systems - Spring 2026

## 📌 Project Overview
Zenith OS is a Terminal User Interface (TUI) simulation designed to visualize high-level Kernel operations. It focuses on the three pillars of Operating Systems: Process Management, Resource Allocation, and Fault Tolerance.

## 🛠️ Tech Stack
- **Language:** Python 3.9
- **Library:** Colorama (for ANSI color-coded system feedback)
- **Environment:** macOS / ZSH Terminal

## 🚀 Key Features
- **Process Life Cycle Visualization:** Real-time state transitions (READY -> RUNNING -> BLOCKED).
- **Resource Simulation:** Dynamic bars tracking CPU Load and Memory consumption.
- **Deadlock Detection:** A simulated "Circular Wait" scenario resulting in a Kernel Panic.
- **Audit Logging:** Live kernel logs tracking every system interrupt.

## 🚦 How to Run
1. Install dependencies: `pip3 install colorama`
2. Execute the simulator: `python3 simulator.py`
